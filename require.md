本專題要求設計並實作一個 Small-C 語言的互動式解譯器 (interactive interpreter) 。此解譯器應提供類似早期 BASIC 解譯器的操作環境：使用者可在命令提示符下逐行輸入 Small-C 程式碼，由解譯器即時解析並執行；亦可透過內建指令載入完整的原始碼檔案後一次執行 。

本專題的設計意圖在於延伸並整合課堂所學的系統軟體核心技術——包括符號表管理、語彙分析 (lexical analysis)、語法導向處理 (syntax-directed processing)、以及執行期環境的模擬——同時引導學生跨越從組合語言處理器到高階語言處理器的認知門檻 。學生在 SIC/XE 組譯器中已經練習過的 token 辨識、符號表建構與查詢、錯誤偵測與回報等技術，均可直接遷移到本專題的開發過程中 。

選用 Python 3 作為實作語言的考量在於：Python 豐富的字串處理能力與動態資料結構可以讓學生將精力集中於解譯器的核心邏輯設計，而非記憶體管理等底層細節 。此外，Python 本身即為直譯式語言，以直譯式語言來建構另一個直譯式語言的過程，有助於學生深刻理解解譯器的運作原理 。

二、Small-C 語言規範 

本專題所定義的 Small-C 為標準 C 語言 (ISO C) 的嚴格子集，其設計參考 Ron Cain (1980) 與 James Hendrix (1984) 所定義的 Small-C 語言，但為配合教學目的進行了若干調整與明確化 。以下規範為本專題解譯器所必須支援的完整語言定義，任何未在此規範中列出的 C 語言特性，學生均不需實作 。

2.1 資料型別 

Small-C 僅支援以下基本資料型別 ：

* 
**int**：有號整數，寬度為 4 個位元組 (32 位元)，數值範圍為 -2,147,483,648 至 2,147,483,647 。


* 
**char**：有號字元，寬度為 1 個位元組 (8 位元)，數值範圍為 -128 至 127，同時可用於儲存 ASCII 字元 。


* 
**int*** 與 **char***：分別為指向 int 與 char 的指標型別 。指標值在解譯器內部以模擬位址 (整數偏移量) 表示 。


* Small-C **不支援** float、double、long、short unsigned、void (但 void 可出現於函式回傳型別，表示無回傳值)、struct、union、enum、typedef，亦不支援多層指標 (如 int) 。



2.2 常數與字面值 

* 
**整數常數**：支援十進位表示法 (如 42、-7) 與十六進位表示法 (以 0x 或 0X 為前綴，如 0xFF) 。


* 
**字元常數**：以單引號包圍 (如 'A'、'0')，並支援以下跳脫序列：`\n` (換行)、`\t` (水平定位)、`\0` (空字元)、`\\` (反斜線)、`\'` (單引號)、`\"` (雙引號) 。


* 
**字串常數**：以雙引號包圍 (如 "hello\n")，在記憶體中以 char 陣列方式儲存，並自動附加結尾的空字元 `\0` 。字串常數僅可出現於函式呼叫的引數位置 (如 `printf("value = %d\n", x);`) 以及 char 陣列的初始化中 。



2.3 變數宣告 

全域變數 (global variables) 與區域變數 (local variables) 的宣告語法相同 ：

```c
int x;
int y = 10;
char ch = 'A';
int arr[20];
char str[80];
int *ptr;
char *cp;

```



* 全域變數宣告於所有函式之外，其生命週期為整個程式執行期間 。


* 區域變數宣告於函式本體的起始處 (所有可執行語句之前)，其生命週期為該函式的單次呼叫期間 。


* Small-C 支援區塊內部的變數宣告（如在 if、while 等區塊內部宣告新變數），其作用域僅限於該區塊內。


* 一維陣列的大小可用整數常數或表達式指定（支援可變長度陣列 VLA）。陣列的索引以 0 為起始 。[額外實作] 支援使用大括號 `{}` 進行陣列初始化列表（如 `int a[3] = {1, 2, 3};`）。



2.4 運算子 

依優先順序由高至低排列如下 (同一層級內由左至右結合，除非特別標註為右結合) ：

* 
**第1級(最高)**：後綴運算子 `()` 函式呼叫、`[]` 陣列索引 。


* 
**第2級**：前綴一元運算子 `-` (負號)、`!` (邏輯否定)、`~` (位元補數)、`*` (指標取值)、`&` (取址)、`++` (前綴遞增)、`--` (前綴遞減) 。此級為右結合 。[額外實作] 支援強制轉型 `(type)expr`。


* 
**第3級**：乘法類 `*` (乘)、`/` (除)、`%` (餘數) 。


* 
**第4級**：加法類 `+` (加)、`-` (減) 。


* 
**第5級**：位移運算子 `<<`、`>>` 。


* 
**第6級**：關係運算子 `<`、`<=`、`>`、`>=` 。


* 
**第7級**：相等運算子 `==`、`!=` 。


* 
**第8級**：位元 AND `&` 。


* 
**第9級**：位元 XOR `^` 。


* 
**第10級**：位元 OR `|` 。


* 
**第11級**：邏輯 AND `&&` (支援短路求值) 。


* 
**第12級**：邏輯 OR `||` (支援短路求值) 。


* 
* **第13級(最低)**：指定運算子 `=`、`+=`、`-=`、`*=`、`/=`、`%=`。此級為右結合 。
* [額外實作] **三元運算子**：`condition ? expr1 : expr2`，優先級介於指定運算子與邏輯 OR 之間。[額外實作] 支援強制轉型 `(type)expr`。
(註：後綴 ++ 與 -- 不列入必要支援範圍，學生可自行選擇是否實作 。)



2.5 控制結構 

Small-C 支援以下控制結構，其語意與標準 C 相同 ：

* 
**if / else**：支援 `else if` 的鏈式寫法 。


* 
**while**：迴圈 。


* 
**for**：迴圈，包含初始化表達式、條件表達式與迭代表達式三個部分 。


* 
**do/while**：迴圈 。


* 
**break**：用於跳出最內層的 while、for 或 do/while 迴圈 。


* 
**continue**：用於跳至最內層迴圈的下一次迭代 。


* 
**return**：用於從函式返回，可附帶回傳值 (`return expr;`) 或不帶回傳值 (`return;`) 。


* 
**switch/case**：為選擇性實作的加分項目，不列為必要要求 。



2.6 函式 

函式定義的語法如下 ：

```c
int add(int a, int b) {
    return a + b;
}
void greet() {
    printf("Hello!\n");
}

```



* 函式的回傳型別可為 int、char 或 void 。


* 參數傳遞方式為傳值呼叫 (call by value)，指標參數可用於模擬傳址效果 。


* 函式支援遞迴呼叫 。


* 所有函式必須在呼叫之前完成定義 (不支援前向宣告) 。


* 每個可執行程式必須包含一個名為 `main` 的函式作為程式進入點，其宣告形式為 `int main()` 或 `void main()` 。在互動模式下，若使用者輸入的語句不在任何函式定義之內，解譯器應在全域環境中直接執行該語句 。



2.7 內建函式 

解譯器必須提供以下內建函式，供 Small-C 程式直接呼叫。所有內建函式無需事先宣告即可使用 。

輸入與輸出函式 

* 
`int putchar(int ch)`：輸出一個字元至標準輸出，回傳該字元的整數值 。


* 
`int getchar()`：從標準輸入讀取一個字元，回傳其整數值；若遇到輸入結束，回傳 -1 。


* 
`void printf(char *fmt, ...)`：格式化輸出函式，必須支援的格式碼包括 `%d` (十進位整數)、`%c` (字元)、`%s` (字串)、`%x` (十六進位整數)、`%%` (輸出百分號字元本身) 。不需支援欄位寬度、精確度或對齊旗標等進階格式化功能 。


* 
`void puts(char *s)`：輸出一個字串並自動換行 。


* 
`int scanf(char *fmt, ...)`：格式化輸入函式，必須支援的格式碼包括 `%d` (讀取整數) 與 `%c` (讀取字元)。[額外實作] 同時支援 `%s` (讀取字串) 與 `%x` (讀取十六進位整數)。引數必須為指標 (使用 `&` 取址運算子) 。回傳成功讀取的項目數 。



字串處理函式 

* 
`int strlen(char *s)`：回傳字串的長度 (不含結尾的 `\0`) 。


* 
`void strcpy(char *dest, char *src)`：將來源字串複製到目的字串 。


* 
`int strcmp(char *s1, char *s2)`：比較兩個字串，回傳值為負數、零或正數，分別表示 s1 在字典順序中小於、等於或大於 s2 。


* 
`void strcat(char *dest, char *src)`：將來源字串串接至目的字串的末尾 。



數學函式 

* 
`int abs(int x)`：回傳 x 的絕對值 。


* 
`int max(int a, int b)`：回傳 a 與 b 中的較大值 。


* 
`int min(int a, int b)`：回傳 a 與 b 中的較小值 。


* 
`int pow(int base, int exp)`：回傳 base 的 exp 次方 。當 exp 為負數時回傳 0 (整數除法的自然結果) 。當 exp 為 0 時回傳 1 。


* 
`int sqrt(int x)`：回傳 x 的平方根之整數部分 (即向下取整) 。當 x 為負數時顯示執行期錯誤 。


* 
`int mod(int a, int b)`：回傳 a 除以 b 的餘數，等同於 `%` 運算子，但以函式形式提供 。當 b 為 0 時顯示執行期錯誤 。


* 
`int rand()`：回傳一個介於 0 至 32767 之間的偽隨機整數 。


* 
`void srand(int seed)`：以指定的種子值初始化偽隨機數產生器 。



記憶體與工具函式 

* 
`void memset(char *ptr, int value, int size)`：將 ptr 所指向的記憶體區塊之前 size 個位元組全部設定為 value 。


* 
`int sizeof_int()`：回傳 int 型別的位元組數 (固定為 4) 。


* 
`int sizeof_char()`：回傳 char 型別的位元組數 (固定為 1) 。


* 
`int atoi(char *s)`：將字串轉換為整數 。若字串不包含有效的整數表示，回傳 0 。


* 
`void itoa(int value, char *str)`：將整數轉換為十進位字串表示，儲存至 str 所指向的字元陣列 。


* 
`void exit(int code)`：立即終止程式執行，回傳指定的結束碼 。



2.8 註解 

支援 C 風格的區塊註解 `/*...*/` 以及 C++ 風格的單行註解 `//...` 。區塊註解不支援巢狀 。

2.9 前處理器 

僅需支援 `#define` 指令的簡單常數定義形式，例如：

```c
#define MAX_SIZE 100
#define PI_APPROX 3

```

不需支援帶參數的巨集定義 (如 `#define MAX(a,b) ((a)>(b)?(a):(b))`)，亦不需支援 `#include`、`#ifdef`、`#ifndef`、`#endif` 等條件編譯指令 。[額外實作] 支援遞迴/嵌套的巨集展開。

三、解譯器環境互動指令 

解譯器啟動後應顯示歡迎訊息與版本資訊，隨後進入互動模式，以提示符 `sc>` 等候使用者輸入 。使用者的輸入分為兩類：若輸入以保留指令關鍵字起始，解譯器執行對應的環境操作；否則將輸入視為 Small-C 程式碼，進行解析與執行 。所有環境指令均不區分大小寫 (case-insensitive) 。以下為必備環境指令：

3.1 程式管理指令 

* 
**LOAD <filename>**：從指定檔案載入 Small-C 原始碼至程式緩衝區 。載入成功後顯示讀取的行數。若檔案不存在或無法開啟，應顯示適當的錯誤訊息。載入動作會覆蓋目前程式緩衝區的內容，執行前應提示使用者確認是否放棄未儲存的修改 。


* 
**SAVE <filename>**：將目前程式緩衝區的內容儲存至指定檔案 。儲存成功後顯示寫入的行數。若檔案無法建立或寫入，應顯示適當的錯誤訊息 。


* 
**LIST**：列出目前程式緩衝區的完整內容，每一行前方標示行號。若緩衝區為空，應顯示提示訊息 。


* 
**LIST <n>**：列出程式緩衝區中第 n 行的內容 。


* 
**LIST <n1>-<n2>**：列出程式緩衝區中第 n1 行至第 n2 行的內容 。


* 
**EDIT <n>**：將第 n 行的內容顯示給使用者，並允許使用者輸入新內容來取代該行。若使用者直接按下 Enter 而不輸入任何內容，則保留原行不變 。


* 
**DELETE <n>**：刪除程式緩衝區中的第 n 行，其後各行的行號自動遞減 。


* 
**DELETE <n1>-<n2>**：刪除程式緩衝區中第 n1 行至第 n2 行的內容 。


* 
**INSERT <n>**：在第 n 行之前進入插入模式，使用者可逐行輸入新的程式碼，直到輸入單獨一行 `.` (句點) 為止，結束插入模式 。


* 
**APPEND**：在程式緩衝區的末尾進入插入模式，使用者可逐行輸入新的程式碼，直到輸入單獨一行 `.` (句點) 為止 。


* 
* **NEW / RESET**：清除程式緩衝區的所有內容，同時重置所有全域變數、函式定義與執行期狀態，回到初始狀態。若目前緩衝區有未儲存的修改，應提示使用者確認 。[額外實作] 支援 RESET 作為別名。



3.2 執行與除錯指令 

* 
**RUN**：執行目前程式緩衝區中的程式 。解譯器依序進行詞法分析、語法分析與語意檢查，若無錯誤則從 `main()` 函式開始執行。若程式中未定義 `main()` 函式，應顯示錯誤訊息。執行結束後顯示程式的回傳值 (若 main 回傳型別為 int) 。`RUN` 指令在執行前應自動清除前一次執行所殘留的全域變數值與動態狀態，確保每次執行都從乾淨的初始狀態開始 。


* 
**CHECK**：對目前程式緩衝區中的程式碼進行語法與語意檢查，但不實際執行 。將所有偵測到的錯誤與警告依行號順序列出。若無錯誤，顯示「No errors found」 。


* 
**TRACE ON**：開啟執行追蹤模式 。啟用後，`RUN` 指令在執行過程中將於每個語句執行前顯示該語句所在的行號與語句內容，格式為 `[Line n] <statement>`。此功能有助於學生除錯 。


* 
**TRACE OFF**：關閉執行追蹤模式 。


* 
**VARS**：顯示目前所有全域變數的名稱、型別與當前值 。若為指標，同時顯示其所指向的位址值。若為陣列，顯示陣列長度及前十個元素的值 (若元素超過十個，以 ... 表示省略) 。


* 
**FUNCS**：列出目前已定義的所有函式名稱、回傳型別、參數列表及其起始行號。內建函式以 `[built-in]` 標示 。



3.3 系統指令 

* 
**HELP**：顯示所有可用環境指令的摘要說明 。


* 
**HELP <command>**：顯示指定環境指令的詳細說明與使用範例 。


* 
**ABOUT**：顯示解譯器的名稱、版本號、作者資訊與修課學期 。


* 
**CLEAR**：清除終端機畫面 (等同於執行系統的清屏操作) 。


* 
**QUIT 或 EXIT**：結束解譯器。若目前程式緩衝區有未儲存的修改，應提示使用者確認是否放棄修改後離開 。

四、執行範例 

(註：以下範例模擬學生以 Bottom-Up 方法逐步開發測試的過程 。)

**範例1：基本算術表達式 (驗證表達式求值引擎)** 

```text
Small-C Interactive Interpreter v3.0
System Software Final Project, Spring 2026
Type 'HELP' for a list of commands.
sc> printf("%d\n", 2+3*4);
14
sc> printf("%d\n", (2+3)*4);
20
sc> printf("%d\n", 100/7);
14
sc> printf("%d\n", 100 % 7);
2
sc> printf("%d\n", -5+3);
-2
sc> printf("%d\n", 2+3*4-8/2);
10
sc> printf("%d\n", ((2+3)*(4-1))/5);
3

```

**範例2：關係與邏輯運算 (驗證布林求值與短路邏輯)** 

```text
sc> printf("%d\n", 5 > 3);
1
sc> printf("%d\n", 5 < 3);
0
sc> printf("%d\n", 5 == 5);
1
sc> printf("%d\n", 5 != 5);
0
sc> printf("%d\n", 5 >= 5 && 3 < 4);
1
sc> printf("%d\n", 5 > 10 || 3 < 4);
1
sc> printf("%d\n", !(5 > 3));
0

```

**範例3：位元運算 (驗證位元層級操作)** 

```text
sc> printf("%d\n", 0xFF & 0x0F);
15
sc> printf("%d\n", 0xA0 | 0x05);
165
sc> printf("%d\n", 0xFF ^ 0x0F);
240
sc> printf("%d\n", ~0);
-1
sc> printf("%d\n", 1 << 8);
256
sc> printf("%d\n", 256 >> 4);
16
sc> printf("0x%x\n", (0xAB & 0xF0) | 0x0C);
0xac

```

**範例4：變數宣告與指定 (驗證符號表與記憶體管理)** 

```text
sc> int x=10;
sc> int y=20;
sc> int z;
sc> z = x + y;
sc> printf("x=%d, y=%d, z=%d\n", x, y, z);
x=10, y=20, z=30
sc> x += 5;
sc> y -= 3;
sc> printf("x=%d, y=%d\n", x, y);
x=15, y=17
sc> char ch='A';
sc> printf("ch=%c (ASCII=%d)\n", ch, ch);
ch=A (ASCII=65)
sc> VARS
int x=15
int y=17
int z=30
char ch=65('A')

```

**範例5：數學內建函式 (驗證內建函式呼叫機制)** 

```text
sc> printf("abs(-42)=%d\n", abs(-42));
abs(-42)=42
sc> printf("max(10, 25) = %d\n", max(10, 25));
max(10, 25)=25
sc> printf("min(10, 25) = %d\n", min(10, 25));
```
